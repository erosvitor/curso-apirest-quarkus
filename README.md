## Sobre
Desenvolvimento de APIs REST utilizando Quarkus e linguagem Java.

## Pré-requisitos
* JDK 17
* Maven 3.8.x
* MySQL 8
* Postman

## Projeto
O projeto desenvolvido no curso será uma API para gerenciamento de filmes.
